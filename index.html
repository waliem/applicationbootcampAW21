What is an API?

API stands for Application Programming Interface. It is the system that we communicate
with when we put in requests through our applications or websites. We can’t talk to the internal system in the back
aka servers or databases. we can only communicate with the API layer. Let’s say you want to know what the weather is
gonna be in Gothenburg tomorrow so you open your weather application. The weather application
sends a request through the internet to an API requesting a weather prognosis for tomorrow in the chosen city.
the API furthers the request to the the server or database that stores the weather prognosis information.
once the information has been located it will output the weather information back through
the API and appear in your application with tomorrows weather.
